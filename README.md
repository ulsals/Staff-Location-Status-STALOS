STALOS merupakan aplikasi berbasis lokasi yang dirancang untuk memantau status keberadaan Kepala Jurusan Teknik Informatika secara real-time. Aplikasi ini membantu dosen, staf, maupun mahasiswa untuk mengetahui apakah Kepala Jurusan sedang berada di Kampus Indralaya, Kampus Palembang, Kampus Bukit, atau sedang berada di luar lokasi kampus.
